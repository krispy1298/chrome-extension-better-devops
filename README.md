# Better Azure Boards Chrome Extension

A Chrome extension to enhance your Azure Boards workflow and productivity.

## Features

- Improves layout and readability
- Easy to install and use

## Developer Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/krispy1298/chrome-extension-better-devops.git
   ```

2. Run `npm i && npm run build`

3. Open Chrome and navigate to `chrome://extensions/`

4. Enable "Developer mode" in the top right corner

5. Click "Load unpacked" and select the `/dist` folder

## Usage

After installation, the extension will be available in your Chrome toolbar. Click the icon to access its features.

## Development

To contribute or modify this extension:

1. Make your changes to the source files
2. Run `npm run build`
3. Reload the extension in `chrome://extensions/`
4. Test your changes thoroughly
